---
title: "Supported Chains"
api: "GET https://api.mintlify.com/supportedChains"
description: "Returns all chains supported by Transfer."
---

### Response

The response will be an array of Chain objects. Each Chain object has the following structure:

<ResponseField name="blockExplorer" type="string">
  The URL of the block explorer for the chain.
</ResponseField>

<ResponseField name="chainId" type="int">
  The unique identifier for the chain.
</ResponseField>

<ResponseField name="isTestnet" type="boolean">
  Indicates whether the chain is a testnet. True if it is a testnet, false
  otherwise.
</ResponseField>

<ResponseField name="logoURI" type="string?">
  The URI for the chain's logo. This field is optional and might not always be
  present.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the chain.
</ResponseField>

<RequestExample>
```bash Example Request
curl --location --request GET 'https://api.mintlify.com/supportedChains' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API_TOKEN}' \
```

</RequestExample>
