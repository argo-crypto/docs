---
title: "Supported Tokens"
api: "GET https://api.mintlify.com/supportedTokens"
description: "Returns all the tokens supported by Transfer. If a chainId is provided, it returns all the supported tokens in that chain."
---

<ParamField query="chainId (optional)" type="int">
  The unique identifier for the chain. If provided, the response will return all
  the supported tokens in that chain.
</ParamField>

### Response

The response will be an array of Token objects. Each Token object has the following structure:

<ResponseField name="address" type="string">
  The address of the token.
</ResponseField>

<ResponseField name="chainId" type="int">
  The unique identifier for the chain.
</ResponseField>

<ResponseField name="decimals" type="int">
  The number of decimals for the token.
</ResponseField>

<ResponseField name="logoURI" type="string?">
  The URI for the token's logo. This field is optional and might not always be
  present.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the token.
</ResponseField>

<ResponseField name="symbol" type="string">
  The symbol of the chain.
</ResponseField>

<RequestExample>
```bash Example Request
curl --location --request GET 'https://api.mintlify.com/supportedTokens?chainId=5' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API_TOKEN}' \
```

</RequestExample>
